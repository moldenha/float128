# Header Only Constexpr Float128
This is a constexpr `C++17` and up 128 bit floating point. It has constexpr capabilities, header only, and certain math functions implemented. This comes directly from the [NeuroTensor](https://github.com/moldenha/NeuroTensor) framework.

## Usage:
- Can be called with the `nt::float128_t` type.
